# 改造记录

## 目标
MSPM0G3507 八路灰度循迹小车，将传感器 ADC 读取改为 **DMA burst 搬运**。

## 仓库
GitHub: https://github.com/giaoLRL/simple-standard-car  
（仓库 slug 为英文 `simple-standard-car`，描述为中文“简易标准”）

## 当前状态
- ADC 配置为 repeat-single-channel 模式。
- 每路传感器启动 1 次 DMA，连续搬运 8 个样本到 `g_adc_dma_buf[8]`。
- DMA 完成中断仅触发 1 次，CPU 在中断里停止 ADC 并置标志。
- MUX 切换仍由 CPU 控制（硬件只有 1 个 OUT）。

## SysConfig 配置

1. **ADC repeat 模式**：`LINE_ADC.repeatMode = true`。
2. **DMA 触发源**：ADC0 MEM0 结果加载事件 `DL_ADC12_DMA_MEM0_RESULT_LOADED`。
3. **DMA 搬运方式**：源地址固定、目标地址递增、半字宽、`transferSize = 8`。
4. **PB21 按键**：输入、内部上拉、PORTB Pin 21。
5. **PA2 蜂鸣器**：Standard 推挽输出、初始低电平、PORTA Pin 2。
6. **SDK 2.10 API**：DMA 外设、中断和通道分别使用 `DMA`、`DMA_INT_IRQn`、`LINE_DMA_CHAN_ID`；完成事件为 `DL_DMA_EVENT_IIDX_DMACH0`。

## 已知限制

- MUX 切换仍是 CPU 阻塞操作。
- DMA burst 完成后才唤醒主循环，本质仍是阻塞式，但中断次数从 64 次/周期降到 8 次/周期。
- 已移除运行中动态学习；黑/白基准由 PB21 按键触发上电双标定确定。
- 丢线时停车，蜂鸣器以 1 Hz 间歇鸣叫报警。
- 支持左/右直角弯状态机，检测到横向黑线后差速转向，中心探头重新找线后恢复。
- DMA 忙等带有限超时和外设恢复，触发异常不会永久卡死。
- 按手册严格要求白大黑小；反极性或跨度不足时用三声提示要求重新标定。
- 归一化在转换为 `uint16_t` 前限幅，避免亮区数值回绕为黑色。
- 转弯超时和蜂鸣器周期使用 SysTick 毫秒时基，不再依赖主循环执行次数。
- `uart_debug_printf` 的 640 字节缓冲已从栈移到 `static`，避免主循环内发送 HELLO 时因调用链 + 中断嵌套导致栈溢出 HardFault。
